## Webpage for Pizzeria Heyday

A high-performance, visually striking web presentation for Pizzeria Heyday located in Snina, Slovakia. This project focuses on a premium aesthetic, seamless user experience, and mobile-first conversion.


## Tech Stack

- Framework: Next.js (App Router)

- Styling: Tailwind CSS

- Animations: Framer Motion

- Language: TypeScript



# Installation 

```bash
npm install
npm run dev
```
